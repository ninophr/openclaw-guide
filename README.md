# OpenClaw Guide — Landing Page

Free guide: "Démarrer avec OpenClaw en 30 minutes"

## Deploy

### To Vercel
1. `git init && git add . && git commit -m "initial"`
2. Push to GitHub: `git push origin main`
3. Import on Vercel: https://vercel.com/new
4. Done — your site is live

### Locally
```bash
npm start
# Open http://localhost:3000
```
